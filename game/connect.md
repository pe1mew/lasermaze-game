# Connection details of laser emitters and detectors

## emitter

```
            Wire connector
   +---------+-----+
   |         |  0  | negative (-) ground
+-------+    +-----+
| laser |    |  0  | 
+-------+    +-----+
   |         |  0  | positive (+) VCC
   +---------+-----+

```

## detector


```
            Wire connector
   +---+---+-+-----+
   |   | P | |  0  | positive (+) VCC
  /-\  +---+ +-----+
 |LDR|       |  0  | negative (-) ground
  \-/        +-----+
   |         |  0  | output 
   +---------+-----+

```
